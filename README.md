## 安装

```shell
pip install openctp-tts==6.3.15.*
pip install openctp-tts==6.3.19.*
pip install openctp-tts==6.5.1.*
pip install openctp-tts==6.6.1.*
pip install openctp-tts==6.6.7.*
pip install openctp-tts==6.6.9.*
```

## 功能

- 支持多版本 TTS
    - 6.3.15_20190220
    - 6.3.19_P1_20200106
    - 6.5.1_20200908
    - 6.6.1_P1_20210406
    - 6.6.7_20220304
    - 6.6.9_20220920
- 支持多版本 Python (3.7 ~ 3.11)
- 支持多平台
    - Windows x64
    - Linux x64
    - Mac x64 arm64
